# Filtering in PCL

These examples are related to filtering in PCL.
For more info, see `pcl_guide.txt` or
[Filtering in PCL](https://pcl.readthedocs.io/projects/tutorials/en/latest/index.html#filtering).

List of examples:

1. `passthrough.cpp`: cut off values that are either inside or outside a given user range; see [Passthrough Filtering in a PointCloud](https://pcl.readthedocs.io/projects/tutorials/en/latest/passthrough.html#passthrough).
